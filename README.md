# 🧠 EEG Schizophrenia Detection using Machine Learning

An AI-powered system that analyzes EEG brain signals to detect patterns associated with schizophrenia using machine learning and signal processing techniques.

This project provides an interactive web interface where users can upload EEG data and receive predictions along with signal visualization.

---

## 📌 Project Overview

Schizophrenia is a serious mental disorder that affects how a person thinks, feels, and behaves. EEG (Electroencephalography) signals capture brain electrical activity and can reveal neurological abnormalities.

This project uses machine learning to analyze EEG signals and detect abnormal patterns related to schizophrenia.

---

## 🚀 Features

- Upload EEG signal data (CSV format)
- Automatic feature extraction from EEG signals
- Machine learning based prediction
- EEG signal visualization
- Interactive web interface using Streamlit
- Real-time prediction results
